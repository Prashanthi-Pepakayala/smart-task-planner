\# Smart Task Planner



A full-stack task planner application built with HTML, CSS, JavaScript (frontend), Node.js \& Express (backend), and MongoDB (database).



---



\## Features



\- Add tasks with priority (High, Medium, Low)

\- Mark tasks as completed

\- Delete tasks

\- Persistent storage using MongoDB

\- Full CRUD operations via backend API

\- Clean UI with animations

\## How to Run Locally



\### 1. Clone the repository

```bash

git clone https://github.com/Prashanthi-Pepakayala/smart-task-planner.git

cd "Smart Task Planner"


2. Backend Setup

cd Backend

npm install

Create a .env file:



MONGO\_URI=your\_mongodb\_connection\_string

PORT=5000



Start the server:



node server.js



Backend runs at:



http://localhost:5000/tasks



3\. Frontend Setup



Simply open:



Frontend/index.html
